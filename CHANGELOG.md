# CHANGELOG

이 문서는 "예산 검증 도구"의 주요 변경 이력을 담습니다.
사내 소규모 도구 특성상 엄격한 시맨틱 버저닝 대신, 굵직한 변경 단위로만 기록합니다.

## v3 (현재 배포 버전 · 2026-07-18)

### 추가
- **RCMS 지원 추가** — PMS·이지바로 2개 시스템만 비교하던 것을 RCMS까지 확장, 참여기관 블록을 자동 식별해 집계
- 여러 과제를 한 번에 검증하는 배치 모드 — 파일이 여러 개 인식되면 직접 짝을 만들어 한 번에 전체 검증
- **검증 결과 요약 표** — 여러 과제를 한 화면에서 한눈에 비교 (1건일 때도 항상 표시)
- **짝짓기 오류 자동 감지** — 실행예산 총액이 비정상적으로 차이나거나 겹치는 비목이 없으면 "혹시 다른 과제 파일 아닐까요?"라고 별도 표시
- 이미 다른 짝에 사용된 파일은 선택 목록에서 자동 제외
- RCMS 원본 "사용잔액"과 재계산 값 교차 검증
- 분류표에 없는 세목은 "기타(미분류)"로 모아 화면에 안내
- 결과를 과제별 시트로 나눈 엑셀 다운로드
- 사용법 아코디언에 PMS·이지바로·RCMS별 실제 다운로드 경로 안내 추가
- 전화요금 계정분할 자동화 도구와 톤앤매너 통일 (헤더·푸터·버튼 스타일, 마이크로 인터랙션)
- GitHub Pages 정적 배포 지원 (`Content-Security-Policy` 적용)

### 수정
- 코드 구조를 단일 파일에서 계산 로직(`core`) · 시스템별 어댑터(`systems`) · 설정(`config`) · 화면(`ui`)으로 분리해 유지보수성 개선
- 파일명 등 화면에 표시되는 값 전체에 이스케이프 처리 추가 (특수문자가 포함된 파일명으로 인한 문제 방지)
- 좁은 화면에서 "짝 추가" 버튼이 비정상적으로 늘어나던 문제 수정
- 요약 표 헤더 정렬 · 줄바꿈 문제 수정, 첫 열 표기를 "#"에서 "순번"으로 변경
- 짝짓기 오류 감지 기준을 실행예산 합계 차이로 단순화 (집행실적은 시점 차이로 원래 변동이 있어 판정 기준에서 제외)
- 키보드 포커스 표시, 스크린리더 안내 텍스트 등 접근성 개선

## v2 (RCMS 이전 · 이지바로 전용 버전)

- 기존 엑셀 매크로(VBA)의 계산 로직을 웹으로 이식 (파일 내용 기반 자동 인식 → 헤더 텍스트로 컬럼 탐색)
- 알파벳 컬럼 위치에 의존하던 매크로 방식을 헤더 기반 탐색으로 교체해, 양식이 바뀌어도 안전하게 동작하도록 개선
- 미분류 세목을 조용히 누락시키지 않고 "기타(미분류)"로 표시하도록 개선
- 결과 엑셀 다운로드를 화면 스크래핑 방식에서 계산 데이터 기반 방식으로 재작성
- SheetJS 버전에 따라 병합 셀 읽는 방식이 달라 계산이 깨지던 문제 수정

## v1 (엑셀 매크로, 사내 배포 전 단계)

- VBA 매크로 기반 예산 대조 도우미봇 (`.xlsm`)
- 실제 사용 중 발견된 문제: 파일 식별이 업로드 순서에만 의존, 컬럼 위치가 하드코딩되어 있어 양식이 바뀌면 오류 발생
