🔒 업로드한 파일은 서버로 전송되지 않고 이 브라우저 안에서만 처리됩니다.

📖 사용법 펼쳐보기
  1. 파일 업로드 : PMS(ETRIware)와 외부시스템(이지바로, RCMS) 예산 엑셀 파일을 한 번에 끌어다 놓으세요. 자동으로 어떤 파일인지 구분합니다.
  2. 여러 과제를 한 번에 검증하고 싶다면 여러 파일을 함께 올리세요. PMS(ETRIware)·외부시스템 파일이 각각 1개씩만 인식되면 바로 진행되고, 여러 개가 인식되면 화면에 나타나는 선택창에서 [짝 추가]로 원하는 조합을 직접 만들 수 있습니다.
  3. 검증 실행 버튼을 누르면 비목별로 이월예산·실행예산·집행실적·잔액을 비교합니다.
  4. 결과 확인 : 값이 다른 셀은 노란 배경에 빨간 글씨로 표시됩니다. 분류표에 없는 세목은 "기타(미분류)" 행에 합산됩니다.
  5. 내보내기 : [전체 엑셀 다운로드]로 과제별 시트가 나뉜 엑셀 파일을 다운로드 할 수 있습니다.

어디서 다운로드하나요?

  • PMS(ETRIware) : 업무관리 > 예산관리 > 현황관리 > 협약사업예실대비표 > 과제선택 > "관련예산 예산집행현황" 선택 > [엑셀다운] 버튼
  • 통합이지바로 : 집행 > 연구비 사용 현황 > 예산실적대비표 > 우측상단 [인쇄] 버튼 > 좌측상단 [저장] > 파일형식 Microsoft Excel File(*.xlsx)로 저장
  • RCMS : 연구개발기관 > 연구비현황 > 연구비사용현황 > 과제선택 > 우측상단 [엑셀다운로드] 버튼 클릭
